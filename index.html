How does JavaScript work ?
Every time we run a JavaScript program, our code is first adopted by the "parser" inside the JS engine. Once the abstract syntax tree is created by the parser, the JavaScript engine converts the JavaScript code into machine code. When a program written in JavaScript is converted to machine language, the converted code is sent to the system for execution, and finally, that byte code is driven by the system. The parser's job is to test the JavaScript code in a line-by-line manner for syntactic errors because JavaScript is an explanatory scripting language, so whenever an error is detected by the parser it throws up a kind of error. And stop executing the code.
